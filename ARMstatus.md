# :test_tube: ARM support for BEAMinject
Since version 0.3.0, BEAMinject has added support for ARM and ARM64 platforms!

### :hammer_and_wrench: ARMv7 support
:warning: **WARNING:** ARMv7 support is pretty much experimental, due to lack of Python runtimes and testing platforms. Use at your own risk.

## :zap: Usage
You need to install a Python runtime and then install dependencies and run manually:
```bat
pip install -r requirements.txt
py BEAMinjector.py
```
For the GUI version, use `py app.py` instead.
